# Deep Learning: Foundations and Concepts

Anki flashcards based on "Deep Learning: Foundations and Concepts" by Christopher Bishop and Hugh Bishop.

## Content

Cards are organized in lesson files under `content/`:
- Each lesson covers a chapter or major topic from the book
- Cards focus on intuition and understanding, not just memorization
- LaTeX formatting for mathematical equations

## Card Format

```json
{
  "uid": "deep-learning-foundations-and-concepts-01-001",
  "front": "Question text with \\( LaTeX \\)",
  "back": "Answer text",
  "tags": ["ch01", "introduction"]
}
```

See [anki-decks](https://github.com/FlyingPumba/anki-decks) for setup and sync instructions.
